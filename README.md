# 📈 프로젝트 목표


- `Model을 ResponseEntity 응답`
- `ResponseEntity ORM`
- `JSON으로 변환`
- `JUnit 테스트`
- `AOP(유효성검사)`
- `JWT`
- `MVC 패턴을 사용한 SSR 구현`
- `MVVM 패턴을 이용한 Flutter 서버 통신`
- `통합 테스트 (API 문서 (RestDoc))`

# 🛠️ 브랜치 전략

- 각 이슈 발생 시 `새로운 브랜치`를 생성
- 작업을 완료 후 Merge 전 `코드 리뷰 진행` 및 `피드백`
- 브랜치 명은 각 인원별 `패키지/기능/이니셜`로 컨벤션을 맞춰 진행
